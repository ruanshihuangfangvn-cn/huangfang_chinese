Cách đưa file này lên GitHub Pages

Tạo repository mới trên GitHub

Ví dụ đặt tên repo là huangfang-chinese-demo.

Upload file này

Đổi tên file thành index.html nếu chưa đúng, rồi upload vào repository.

Bật GitHub Pages

Vào Settings → Pages → Deploy from branch → chọn branch main và folder /root.

Mở link công khai

GitHub sẽ tạo link dạng https://username.github.io/huangfang-chinese-demo/.
